A premium, Dubai-inspired luxury e-commerce website built using **HTML, CSS, and Vanilla JavaScript**. This project showcases a high-end shopping experience focused on **luxury watches and Khanzu (Thobes)**, emphasizing elegance, interactivity, and modern UI/UX design.

---

## ✨ Project Overview

Desert Crown Luxury is a fully responsive front-end e-commerce platform designed to replicate the **opulent lifestyle of Dubai**. The system allows users to browse products, view detailed product pages, add items to a cart, and simulate a checkout process.

This project was built as both:
- 🎓 An academic submission  
- 💼 A portfolio project demonstrating frontend development skills  

---

## 🎯 Objectives

- Build a multi-page e-commerce system without using frameworks
- Create a visually appealing luxury UI
- Implement dynamic functionality using JavaScript
- Simulate real-world e-commerce features (cart, checkout, filtering)
- Practice clean code organization and responsive design

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the web pages  
- **CSS3** – Styling, layout, animations, responsiveness  
- **JavaScript (Vanilla)** – Dynamic functionality and interactivity  
- **LocalStorage API** – Persistent cart system

## 🚀 Features

### 🛍️ E-commerce Functionality
- Product listing with dynamic rendering
- Product detail view with image gallery
- Add to cart system using LocalStorage
- Update cart quantity and remove items
- Dynamic total price calculation
- Simulated checkout process

### 🎨 UI/UX Features
- Luxury-inspired design (dark theme + gold accents)
- Fully responsive (mobile, tablet, desktop)
- Sticky navigation bar
- Smooth scrolling effects
- Hover animations
- Toast notifications (e.g., “Item added to cart”)
- Modal popup (quick product preview)

### ⚙️ Advanced Features
- Product filtering (category, price)
- Product sorting (low to high, high to low)
- Dark/Light mode toggle
- Loading animations

---

## 🧠 How It Works

### 1. Product Data
Products are stored in a JavaScript array (`products.js`) and dynamically rendered on the shop page.

### 2. Cart System
- Items are added to the cart using JavaScript
- Cart data is stored in **LocalStorage**
- Cart updates in real-time across pages

### 3. Navigation
Each page is linked through a consistent navigation bar for seamless user experience.

## ⚠️ Limitations

- No backend (frontend-only project)
- No real payment integration
- No user authentication system
- Data is not stored in a database (uses LocalStorage)
